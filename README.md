## 安装

- Windows

```bash
git clone https://github.com/wbthomason/packer.nvim "$env:LOCALAPPDATA\nvim-data\site\pack\packer\opt\packer.nvim"
...

```

- linux

```bash
mv ~/.config/nvim ~/.config/nvim.bk
git clone https://gitee.com/liranux/nvea.git ~/.config/nvim
nvim + PackerSync
```
